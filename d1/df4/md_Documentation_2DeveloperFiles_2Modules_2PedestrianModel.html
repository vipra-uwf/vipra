timestep(const VIPRA::PedestrianSet&, const VIPRA::ObstacleSet&, const VIPRA::Goals&, VIPRA::State&, VIPRA::delta_t, VIPRA::t_step) -> void

Calculates change in pedestrian position/velocity for each timestep of the simulation. Placing the updated values in the passed in VIPRA::State&.

Called at the start of each simulation time step.

Params:

  • const VIPRA::PedestrianSet& : pedestrian set module
  • const VIPRA::ObstacleSet& : obstacle set module
  • const VIPRA::Goals& : goals module
  • VIPRA::State& : out parameter for updated pedestrian positions
  • VIPRA::delta_t : simulated time since last time step (time step size)
  • VIPRA::t_step : current time step number
